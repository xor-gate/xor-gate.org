**Mash scheme**

[Single infusion low temperature mash for a dry beer](https://beerandgardeningjournal.com/mash-method-i/).

| Type          | Time      | Temperature (celsius) |
| ------------- | --------- | --------------------- |
| Mash-in       | -         | 64 degree             |
| Beta amylase  | 1.5 hour  | 65 degree             |
| Flame-out     | -         | 76 degree             |
| Boil          | 60 min    | 100 degree            |

# Fermentables
